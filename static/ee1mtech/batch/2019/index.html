Autumn 2019 Seminar Topics

Faculty Topic Remarks
Siddhartha P Duttagupta Tera Hertz imaging for analysis of quantum dot patterns
Siddhartha P Duttagupta 3-D bio-marker image distortion and mitigation schemes
Siddhartha P Duttagupta Machine learning for rapid pattern recognition of bio-markers
Rajbabu Velmurugan Multichannel speaker separation 1 student, should be interested in speech/audio, signal processing, and machine learning
Bikash Kumar Dey Storing information in DNA

Bikash Kumar Dey Efficient distributed computing for machine learning
Bikash Kumar Dey The jammer vs. codes: efficient communication in the presence of jammers
V R Sule Secure wireless communication The seminar will involve study and understanding aspects and problems involving securing communication at physical layer, code dependent encryption etc. Prospective continuation to M.Tech. project depending on student interest and evaluation of seminar
Gaurav Kasbekar Security Protocols for the Internet of Things 1 student
Jayanta Mukherjee Topics in antenna design
Jayanta Mukherjee Power Amplifier Design
Jayanta Mukherjee Radar Signal Processing
Sibi Raj B Pillai Deep Learning to Decode 1 student, Deep learning has been applied to develop communication decoders which are robust against channel variations. In this seminar, various existing coding schemes and deep learning methods has to be reviewed. Comparisons in terms of computation and performance will be used to further develop efficient learning decoders.
Sibi Raj B Pillai Polar codes for decentralized high speed wireless links 1 student, Polar codes are finding increased applications in 5G and other high speed communication systems. This seminar involves reading about Polar codes, and implementing them for a multiple access system with two users.
Sibi Raj B Pillai Low power wireless networks 1 student, This seminar is on the design an implementation of LoRa wireless technology on software radio.
Nikhil Karamchandani Sequential estimation/inference algorithms for learning tasks such as clustering, ranking etc.
Nikhil Karamchandani Coding for distributed optimization / storage
Nikhil Karamchandani Modeling and analysis of content caching and delivery networks
Prasanna Chaporkar Implementation and optimization of non-3GPP access to 5G core
Vikram M Gadre Current challenges and innovations in radar signal processing This topic is to be pursued in collaboration with my current PhD students working at LRDE. It would be more appropriate for a person from DRDO or defence sponsored background but others are also welcome, if genuinely interested. Normally to be continued to the Dissertation.
Vikram M Gadre Modern techniques and challenges in biometrics The aim here is to explore the tools and techniques underlying biometric signal and image processing, from the point of view of principles and applications in security, authentication and so on. Normally to be continued to the Dissertation.
Saravanan Vijayakumaran Cryptocurrency and Blockchain Technologies Knowledge of basic cryptography is required
Shalabh Gupta Signal Processing Techniques for Data Center Optical Interconnects Operating at 400 Gbps and Beyond: Towards Low Power Practical Implementation Students should apply for this topic only if they are interested in experimental work
